Authentication API Reference

POST /auth/login

Authenticates a user and returns a JWT token.

Request Body

email (string, required): The user's email address.

password (string, required): The user's password.

Response

Returns a JSON object containing the access token and expiry time.

POST /auth/logout

Invalidates the user's current session token.

Request Headers

Authorization (required): Bearer token from login.

GET /auth/verify

Verifies whether a token is still valid.

Returns 200 if valid, 401 if expired or invalid.
